# 🎮 Zona Cerdas - Educational Game Marketplace

**Zona Cerdas** adalah platform *marketplace* interaktif yang menghubungkan kreator (kontributor) game edukasi dengan para pelajar dan guru di seluruh Indonesia. Pengguna dapat menemukan, membeli, dan memainkan berbagai game edukatif, sementara kreator dapat mengunggah serta mengelola karya mereka.

---

## ✨ Fitur Utama

### 👨‍🎓 Untuk Pengguna (User)

- 📚 **Katalog Game**  
  Jelajahi berbagai game edukasi berdasarkan mata pelajaran, jenjang pendidikan, dan kategori.

- 🎮 **Koleksi Game (Library)**  
  Akses semua game yang telah dibeli dengan mudah dalam satu tempat.

- ▶️ **Bermain Langsung**  
  Mainkan game edukasi secara langsung melalui aplikasi tanpa perlu instalasi tambahan.

---

### 👨‍💻 Untuk Kontributor (Kreator)

- 📝 **Manajemen Konten**  
  Unggah game baru, atur harga, perbarui informasi game, dan kelola thumbnail.

- 💰 **Dashboard Keuangan**  
  Pantau transaksi penjualan dan lakukan proses penarikan saldo (*withdraw*) hasil penjualan game.

---

## 🛠️ Teknologi yang Digunakan

### Frontend

- **React.js** (v18) - Library untuk membangun antarmuka pengguna
- **Vite** - Frontend build tool yang cepat dan modern
- **Tailwind CSS** - Framework CSS untuk desain antarmuka
- **React Router DOM** - Manajemen navigasi dan routing halaman
- **Axios** - HTTP Client untuk komunikasi dengan REST API
- **React Icons** - Kumpulan ikon untuk kebutuhan antarmuka

### Backend

- **Java Spring Boot** - RESTful API untuk mengelola logika aplikasi
- **Database Terintegrasi** - Penyimpanan data pengguna, game, dan transaksi

## 🚀 Cara Menjalankan Proyek Secara Lokal

Pastikan **Node.js** sudah terinstal di komputer Anda.

### 1. Clone Repository

```bash
git clone https://github.com/intannz/zonaCerdas
```

### 2. Masuk ke Folder Proyek

```bash
cd zona-cerdas-frontend
```

### 3. Instal Dependencies

Menggunakan **npm**:

```bash
npm install
```

### 4. Jalankan Development Server

Menggunakan **npm**:

```bash
npm run dev
```

### 5. Buka Aplikasi di Browser

Buka alamat berikut pada browser:

```
http://localhost:5173
```
